📤 How to Upload via Google Drive

  1. Visit Google Drive.
  2. Click “New” → “File upload” and select your file.
  3. After upload, right-click and select “Get link”.
  4. Set visibility to “Anyone with the link”.
  5. Copy the link and paste it into the File URL field above.

📌 Note: Only public Google Drive links will work properly for students.
